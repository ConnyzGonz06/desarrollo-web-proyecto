Requerimientos minimos de hardware y software para equipos de clientes con CGL

En este documento podrás conocer los requerimientos de hardware y software necesarios para el correcto funcionamiento de la aplicación.

Requerimientos mínimos de hardware

  • Procesador Core I3 de 2 GHz o superior.
  • 4 GB de memoria RAM o superior.
  • Disco duro de 100 GB o superior.
  • Impresora láser.
  • Impresora de matriz para contabilidad y tesorería (solo para cheques).

Requerimientos de Sistema Operativo

Sistemas operativos Microsoft en versiones para PC o portátil de Windows 8 o superior de 32 o 64 bits, Sistema operativo Mac OS (módulos compatibles) Framework 4.5.

Requerimientos de Office:

Versiones de Microsoft Office 2016 o superior que tengan Word y Excel. Recomendado Microsoft Office 2019,Office 365 para(Aplicaciones para escritorio)

Requerimientos de Microsoft Edge:

Compatibilidad con los usuarios de CGL con navegadores Google Chrome y Microsoft Edge Si aún maneja un marco antiguo o maneja opciones aun con Internet Explorer, si se recomienda el uso de Navegador Microsoft Edge en modo de retrocompatibilidad.

Requerimientos Conexión Internet

Mínimo 5 Mb/s de descarga y 3 Mb/s de carga,Recomendado (10 Mbps de descarga y 5 Mbps de carga) para conocer su conexión actual sobre nuestros canales lo invitamos a realizar los test de velocidad correspondientes.
